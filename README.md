# Overview

Alertive provides secure communication for healthcare teams, allowing users to:

- Send urgent and non-urgent alerts
- Exchange clinical tasks
- Communicate through secure conversations
- Manage role-based inboxes
- Perform electronic handovers
- Track message delivery and responses
- Attach patient context and supporting information

Alertive works across personal and corporate devices while maintaining healthcare-grade security standards. 【1-4af7da】

# First-Time Setup

## Enable Notifications

Alertive requires notification permissions to operate correctly.

When prompted:

- Select **Allow**
- Enable **Critical Alerts**

Critical alerts may override silent mode and lock-screen settings during emergencies. 【1-4af7da】

## Create Your Passcode

Create a unique 4-digit Alertive passcode.

For faster and more secure access you may also enable:

- Face ID
- Touch ID
- Fingerprint authentication
- Other supported biometrics【1-4af7da】

## Sign In

To access Alertive you will need:

- Organisation ID
- Trust Email Address
- Trust Password

These details are provided by your organisation. 【1-4af7da】

## Fast Sign-In

# Roles and Handovers

## What is a Role?

Roles represent responsibilities within the organisation.

Examples:

- ED Consultant
- Bed Manager
- Site Coordinator
- X-Ray Department

Only one user can occupy a role at any given time. 【1-4af7da】

## Acquiring a Role

### iOS

Swipe right-to-left on the role and select **Acquire**.

### Android

Tap the three-dot menu and select **Acquire**.

Once acquired:

- Messages are delivered to you
- You gain access to the role inbox
- You can respond on behalf of the role【1-4af7da】

## Role Status

Roles may have one of two states:

| Status | Meaning |
|----------|----------|
| Occupied | Someone is currently covering the role |
| Empty | Nobody is currently covering the role |

Avoid sending alerts or tasks to empty roles where possible. 【1-4af7da】

## Handovers

Roles can be:

- Given to another colleague
- Taken over from another colleague
- Relinquished at end of shift

This supports safe and effective clinical handovers. 【1-4af7da】
